# Решение проблем

## Проблемы с производительностью

### Приложение медленно работает

**Симптомы:**
- Долгая загрузка интерфейса
- Задержки при вводе текста
- Медленные ответы от AI

**Решения:**

1. **Проверьте интернет-соединение:**
   - Минимальная скорость: 1 Мбит/с
   - Рекомендуемая: 5+ Мбит/с
   - Используйте speedtest для проверки

2. **Очистите кэш приложения:**
   - Android: Настройки → Приложения → Claude Chat → Очистить кэш
   - iOS: Настройки приложения → Очистить кэш
   - Desktop: Настройки → Дополнительно → Очистить кэш

3. **Обновите приложение:**
   - Проверьте наличие обновлений в магазине приложений
   - Текущая версия: 1.3.0

4. **Перезагрузите устройство**

5. **Закройте другие приложения:**
   - Освободите оперативную память
   - Особенно браузеры и видео

### Приложение зависает или вылетает

**На Android:**
1. Очистите кэш и данные
2. Удалите и переустановите приложение
3. Проверьте наличие обновлений системы
4. Освободите место на устройстве (нужно минимум 500 МБ)

**На iOS:**
1. Принудительно закройте приложение (свайп вверх)
2. Перезагрузите iPhone/iPad
3. Обновите iOS до последней версии
4. Переустановите приложение

**На Desktop:**
1. Закройте приложение через диспетчер задач
2. Удалите папку с кэшем:
   - Windows: `%APPDATA%\ClaudeChat\cache`
   - macOS: `~/Library/Application Support/ClaudeChat/cache`
   - Linux: `~/.config/ClaudeChat/cache`
3. Перезапустите приложение

### Долгая отправка сообщений

**Причины:**
- Медленный интернет
- Серверы перегружены
- Большой размер сообщения
- VPN замедляет соединение

**Решения:**
1. Разбейте длинные сообщения на части
2. Отключите VPN (если не критично)
3. Переключитесь на другую сеть (Wi-Fi → мобильная или наоборот)
4. Попробуйте позже, если серверы перегружены

## Проблемы с функциональностью

### Не сохраняется история чатов

**Проверьте:**
1. Настройки → Конфиденциальность → "Сохранять историю" включено
2. Достаточно места на устройстве
3. Разрешения на запись данных (для мобильных)
4. Не используется режим инкогнито/приватный

**Решение:**
1. Включите сохранение истории в настройках
2. Освободите место на устройстве
3. Предоставьте необходимые разрешения

### Не работает экспорт чатов

**Поддерживаемые форматы:**
- TXT (обычный текст)
- JSON (структурированные данные)
- PDF (для печати)
- Markdown (для разработчиков)

**Если экспорт не работает:**
1. Проверьте разрешения на доступ к файлам
2. Убедитесь, что достаточно места
3. Попробуйте другой формат
4. Обновите приложение

### Темная тема не применяется

**На всех платформах:**
1. Настройки → Внешний вид → Выберите тему
2. Варианты:
   - Светлая
   - Темная
   - Системная (следует за системой)

**На iOS:**
- Баг в iOS 17.2 - обновитесь до 17.3+
- Или выключите "Автояркость"

**На Android:**
- Проверьте Android System WebView (обновите до последней версии)

### Не работают уведомления

**Android:**
1. Настройки системы → Уведомления → Claude Chat → Включить
2. Проверьте батарею: не включен ли режим энергосбережения
3. Убедитесь, что приложение не в фоновых ограничениях
4. Разрешите автозапуск (для некоторых производителей)

**iOS:**
1. Настройки → Уведомления → Claude Chat → Разрешить
2. Проверьте "Не беспокоить" / "Режим концентрации"
3. Убедитесь, что звук включен

**Desktop:**
1. Настройки системы → Уведомления → Разрешить для Claude Chat
2. В приложении: Настройки → Уведомления → Включить

## Ошибки и коды

### Ошибка "Network timeout"

**Означает:** Превышено время ожидания ответа от сервера

**Решения:**
1. Проверьте стабильность интернета
2. Перезапустите роутер
3. Отключите VPN
4. Попробуйте другую сеть
5. Проверьте брандмауэр/антивирус

### Ошибка "Rate limit exceeded"

**Означает:** Превышен лимит запросов

**Для Free плана:** 50 сообщений в день
**Для Premium:** Обычно не возникает, но есть защита от спама (100 сообщений в минуту)

**Решения:**
1. Подождите до сброса лимита (в полночь UTC)
2. Обновитесь до Premium для снятия лимитов
3. Не отправляйте сообщения слишком часто

### Ошибка "Invalid API key"

**Причины:**
- API ключ не настроен
- API ключ истек
- API ключ неверный

**Решения:**
1. Настройки → API → Проверьте ключ
2. Получите новый ключ на console.anthropic.com
3. Убедитесь, что скопировали ключ полностью
4. Проверьте, не истек ли срок действия

### Ошибка "Content policy violation"

**Означает:** Сообщение нарушает политику использования

**Что запрещено:**
- Призывы к насилию
- Дискриминация
- NSFW контент
- Спам
- Вредоносный код

**Решение:**
1. Перефразируйте вопрос
2. Убедитесь, что запрос соответствует правилам
3. Если считаете блокировку ошибочной - обратитесь в поддержку

### Ошибка "Server error 500"

**Означает:** Внутренняя ошибка сервера

**Это временная проблема:**
1. Подождите 5-10 минут
2. Попробуйте снова
3. Проверьте статус сервисов: status.example.com
4. Если проблема сохраняется > 1 часа - обратитесь в поддержку

## Проблемы на конкретных платформах

### Android специфичные

**Приложение не устанавливается:**
- Нужна Android 7.0+
- Освободите минимум 100 МБ
- Разрешите установку из Google Play

**Не работает после обновления системы:**
- Переустановите приложение
- Очистите кэш Google Play Services

### iOS специфичные

**Не работает на старых iPhone:**
- Требуется iOS 14.0+
- Для полной функциональности iOS 15.0+

**Проблемы с Face ID/Touch ID:**
- Перенастройте биометрию в системе
- Дайте разрешения приложению

### Desktop специфичные

**Windows:**
- Требуется Windows 10/11
- Установите Microsoft .NET Runtime если просит
- Добавьте в исключения Windows Defender

**macOS:**
- Требуется macOS 11.0+
- При первом запуске: Системные настройки → Защита → Разрешить
- Для Apple Silicon (M1/M2) используйте нативную версию

**Linux:**
- Поддержка: Ubuntu 20.04+, Fedora 35+, Arch
- Установите зависимости: `sudo apt install libgtk-3-0 libwebkit2gtk-4.0`

## Сброс настроек

### Мягкий сброс (сохраняет историю)

1. Настройки → Дополнительно
2. "Сбросить настройки"
3. Подтвердите действие
4. Перезапустите приложение

### Полный сброс (удаляет все данные)

1. Настройки → Дополнительно
2. "Полный сброс"
3. Введите пароль для подтверждения
4. Все данные будут удалены
5. Потребуется заново войти в аккаунт

## Когда обращаться в поддержку

Обратитесь в поддержку, если:
- Проблема не решается описанными способами
- Обнаружили баг
- Потеряли доступ к аккаунту
- Проблемы с оплатой
- Вопросы по безопасности

**Контакты:**
- Email: support@example.com
- Встроенный чат: Настройки → Помощь → Связаться с поддержкой
- Время ответа:
  - Free: до 24 часов
  - Premium: до 2 часов

**При обращении укажите:**
- Версию приложения
- Операционную систему
- Описание проблемы
- Шаги для воспроизведения
- Скриншоты (если возможно)
