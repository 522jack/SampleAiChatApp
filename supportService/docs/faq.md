# Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Что такое Claude Chat?
Claude Chat - это кроссплатформенное приложение для общения с AI-ассистентом Claude. Приложение работает на Android, iOS и Desktop (Windows, macOS, Linux).

### Нужна ли регистрация?
Да, для использования приложения необходима регистрация. Вы можете зарегистрироваться через:
- Email и пароль
- Google OAuth
- Apple ID (на устройствах Apple)

### Сколько стоит использование?
Мы предлагаем два тарифных плана:
- **Free** - бесплатный с ограничением 50 сообщений в день
- **Premium** - 490₽/месяц, неограниченное количество сообщений и приоритетная поддержка

## Функциональность

### Сохраняется ли история чатов?
Да, все ваши разговоры сохраняются автоматически и доступны в разделе "История чатов". Вы можете:
- Просматривать прошлые беседы
- Искать по ключевым словам
- Экспортировать в различные форматы
- Удалять ненужные чаты

### Можно ли использовать приложение офлайн?
Нет, для работы приложения требуется подключение к интернету, так как обработка запросов происходит на серверах Claude API.

### Поддерживается ли темная тема?
Да, приложение поддерживает как светлую, так и темную тему. Настройка доступна в меню Настройки → Внешний вид.

### Как очистить историю чатов?
1. Откройте раздел "История чатов"
2. Выберите чат, который хотите удалить
3. Нажмите на иконку корзины
4. Подтвердите удаление

Для полной очистки: Настройки → Конфиденциальность → Очистить всю историю

## Проблемы и решения

### Приложение медленно работает
Попробуйте следующие решения:
1. Проверьте скорость интернет-соединения
2. Очистите кэш приложения
3. Обновите приложение до последней версии
4. Перезагрузите устройство

### Не приходят уведомления
1. Проверьте настройки уведомлений в системных настройках
2. Убедитесь, что уведомления включены в настройках приложения
3. Проверьте, не включен ли режим "Не беспокоить"

### Ошибка при отправке сообщения
Возможные причины:
- Отсутствует интернет-соединение
- Превышен лимит сообщений (для Free плана)
- Временные проблемы с сервером
- Сообщение содержит запрещенный контент

Решение: проверьте соединение и повторите попытку через несколько минут.

## Безопасность

### Безопасны ли мои данные?
Да, мы используем:
- End-to-end шифрование для передачи данных
- Безопасное хранение API ключей
- Не передаем данные третьим лицам
- Соответствуем GDPR

### Можно ли удалить свой аккаунт?
Да, в любой момент через Настройки → Аккаунт → Удалить аккаунт. При удалении будут удалены все ваши данные и история чатов.

### Используются ли мои сообщения для обучения AI?
Нет, в соответствии с политикой Anthropic, коммерческие данные не используются для обучения моделей.
